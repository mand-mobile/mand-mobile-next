# DropMenu 下拉菜单

下拉菜单可用于列表筛选

# 引入

import { DropMenu } from 'mand-mobile-next'

Vue.createApp().component(DropMenu.name, DropMenu)

参数说明类型默认值

属性说明类型

属性说明类型
